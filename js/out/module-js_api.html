Module: js/api

This is our API. It is essentially the server application which listens for HTTP requests and decides what to do about them, returns responses, etc. The API focuses on HTTP interactions, built on top of Express and Node.js. All HTTP-specific stuff is contained in this module and isolated from the rest. Note that the API needs to include the model and the Data Access Object, and it interacts with both of them to implement functionality
