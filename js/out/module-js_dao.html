Module: js/dao

This is our Data Access Object. It defines functionality for storing information about TodoItems. The rest of our applications only needs to ask the DAO to store something, without caring how it is done. All the database stuff goes in here. This means that only this file needs to contain knowledge about the DB implementation, and it can be easily swapped for another file, using a different database.
